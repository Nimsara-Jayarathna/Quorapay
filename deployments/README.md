# deployments

## Overview
Run and orchestration assets for local prototype execution. Deployment definitions start multiple instances of one shared node codebase with per-instance `NODE_ID`, `PORT`, and `STORAGE_PATH` configuration.

## Contents

- Non-container local startup references.
- Demo environment wiring guidance.
- Service topology definitions for testing.

- Long-term operations tooling.
