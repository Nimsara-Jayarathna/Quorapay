# configs

## Overview
Configuration assets for local and demo environments. Multiple node instances run the same codebase and are differentiated through `NODE_ID`, `PORT`, and `STORAGE_PATH` values.

## Contents
- Node-specific configuration templates.
- ZooKeeper default settings for local coordination.
- Environment-specific parameter examples.
- Non-secret defaults for reproducible runs.

## Not in Scope
- Source code for node services.
- Secrets, credentials, or private keys.
- Deployment automation scripts.
