# cli

## Overview
Command-line tooling for demo execution and lightweight request generation. It should remain simple, scriptable, and focused on repeatable local scenarios.

## Contents
- CLI commands for submitting payments.
- Commands for reading status and ledger data.
- Demo runners for common validation flows.
- Input parsing and command-level validation.

## Not in Scope
- Web UI assets and browser workflows.
- Node-side replication or failover logic.
- Long-lived background services.
