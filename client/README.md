# client

## Overview
Client-facing tools for driving requests and demos against Quorapay nodes. This area contains operator and demo interfaces, not core distributed system internals.

## Contents
- CLI and web client entry points.
- Request construction for payment and ledger operations.
- Client-side validation and formatting helpers.
- Demo-focused client workflows.

## Not in Scope
- Replication, consensus, or storage algorithms.
- ZooKeeper coordination internals.
- Deployment orchestration code.
