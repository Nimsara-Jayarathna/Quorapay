# docs

## Overview
Report-ready project documentation and design artifacts for Quorapay. Shared decisions and evidence belong here to support implementation and evaluation.

## Contents
- Project overview and architecture documentation.
- Protocol, testing, and evaluation records.
- Team-specific working notes under isolated subfolders.
- Small media assets for documentation.

## Not in Scope
- Runtime source code or executable artifacts.
- Large generated datasets or binaries.
- Deployment automation scripts.
