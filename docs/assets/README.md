# assets

## Overview
Small documentation assets used by markdown and report content. Keep files lightweight and directly tied to docs rendering.

## Contents
- Diagrams and figures for architecture/protocol docs.
- Small reference images for test and evaluation sections.
- Source files for reproducible documentation visuals.

## Not in Scope
- Large datasets or generated binaries.
- Runtime application artifacts.
- Unreferenced media files.
